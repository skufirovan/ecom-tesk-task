# Каталог товаров

Одностраничное приложение каталога товаров.

## Стек

- React
- TypeScript
- Vite
- React Router
- CSS Modules
- Prettier / ESLint

## Функциональность

- вывод списка товаров карточками
- поиск товаров по названию
- сортировка по цене
- пагинация
- модальное окно с подробной информацией о товаре
- закрытие модалки по кнопке, клику вне окна и клавише Esc
- имитация API-запросов через Promise с задержкой
- адаптивная верстка для mobile / tablet / desktop

## Сборка

```bash
npm install
npm run build
npm run preview
```

## Структура проекта

src/
  api/          # имитация API-запросов
  components/   # переиспользуемые UI-компоненты
  pages/        # страницы приложения
  shared/       # общие типы, утилиты и константы

## Архитектурные решения

Данные загружаются через отдельный API-слой, чтобы компоненты не зависели напрямую от моковых данных

Поиск, сортировка и пагинация синхронизированы с query-параметрами URL. Это позволяет сохранять состояние страницы при обновлении и использовать навигацию браузера

Для поиска используется библиотека `use-debounce`, чтобы уменьшить количество лишних ререндеров и запросов к API, а также сделать пользовательский опыт более плавным
